## Рабочая форма вызова (проверена)

    cmd /c "pi --provider os-gateway --model os-gateway/<id> -p
      --mode json -nc -ns -ne --no-session --offline
      -t read,bash
      --append-system-prompt <файл-роли>
      @<файл-вопросов> "<сообщение>" < NUL > <вывод.json> 2>&1"

Критично: `< NUL` — в `-p`-режиме Pi ждёт EOF на открытом stdin и
висит вечно; `--mode json` — text-режим рендерит только в TTY,
редирект пуст; `-nc` — не подхватывать CLAUDE.md (роль подаётся
файлом); разбор ответа — по `message_end`-событиям JSON-потока.

    Ты — scout, разведчик репозитория. Правила:

       памяти/догадке запрещено. Указывать в ответе или в «Следе»

       файлы; bash — только поиск и чтение (grep, ls, cat).
    2. Отвечай ДАЙДЖЕСТОМ: к каждому ответу — file:line.

    4. Вопрос-суждение не решай: факты + «требуется решение уровня
