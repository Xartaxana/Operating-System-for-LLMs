ЧТО ВСКРЫЛ ВЕКТОР (рубрика 6а этого не видела):

1. **Дешевизна C частично оплачена невидимым качеством**: по 6а
   C отставал на 1 балл, по F(6а-cost) выигрывал ~45%; вектор
   наказывает нулевые тесты t1 (ось C) и непpersistентность —
   разрыв F сжался до ~10% (10.0 vs 11.1). Табель работает: то,
   «за что политика берёт деньги», стало числом.
2. **Масса тестов ≠ качество тестов**: 37 тестов Bdiet-t1 убивают
   мутации ХУЖЕ (3/4), чем 22 теста Bfull-t1 (5/6) — выжил
   div-by-zero-guard класс. Kill rate ловит то, что счёт тестов
   прячет.
3. Симметричная дыра Bfull-t3/C-t3 (delete-по-позиции выжил: нет
   теста двух последовательных delete) при закрытой в Bdiet-t3 —
   готовый регресс-кейс класса.
4. Итог диеты по вектору: Bdiet 0.91 vs Bfull 0.93 при +$1 —
   равенство в шуме одиночного прогона; вывод «диета не исполнилась
